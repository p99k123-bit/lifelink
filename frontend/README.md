# BloodLine Frontend

Next.js 14 App Router frontend scaffold for BloodLine.

Quick start:

```bash
cd frontend
npm install
npm run dev
```

Environment variables:
- `NEXT_PUBLIC_API_URL` - backend base URL (e.g. https://localhost:5000)
- `NEXT_PUBLIC_USE_HTTP_COOKIE` - set to `true` to ask backend to set HttpOnly cookie after login/signup
